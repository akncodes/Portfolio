## 🛠 Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Animation:** GSAP, Framer Motion
- **Language:** TypeScript

## 🛠 Installation & Set Up

First, install dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
